# Debian Edition
## 1. Instalación de Git
#### Actualizamos el sistema:
```bash
sudo apt update
```
#### Instalamos Git:
```bash
sudo apt install git -y
```
#### Verificamos la instalación:
```bash
git --version
```
## 1. Configuración básica de Git
#### Configuramos la Identidad:
##### Nombre:
```bash
git config --global user.name "TuNombre"
```
##### Correo:
```bash
git config --global user.email "TuCorreo"
```
#### Configuramos el editor por defecto (code, vim, etc...):
##### Code:
```bash
git config --global core.editor "code --wait"
```
##### Vim:
```bash
git config --global core.editor "vim"
```
#### Configuramos la rama por defecto:
```bash
git config --global init.defaultBranch main
```
#### Configuramos color y formato agradable:
```bash
git config --global color.ui auto
```
#### Configuramos pull:
```bash
- git config --global pull.rebase true
```
#### Configuramos push:
```bash
- git config --global push.default simple
```
#### Configuramos el manejo de final de linea (Colaborar con otros SOs):
```bash
git config --global core.autocrlf input
```
#### Verificamos la configuración:
```bash
git config --list --show-origin
```
